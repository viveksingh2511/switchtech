Introduction to Strings

What is a String?

A string in Python is a sequence of characters (letters, digits, symbols, spaces, etc.) enclosed within quotes. Strings are one of the most used data types for storing and manipulating textual data.

You can declare a string using:

  • Single quotes '...'
  • Double quotes "..."
  • Triple quotes '''...''' or """...""" for multi-line strings

Strings can represent names, sentences, identifiers, or even code blocks.

Creating Strings

PYTHON 
s1 = 'Hello'
s2 = "Welcome to Python"
s3 = '''This is
a multi-line
string example'''

print(s1)       # Hello
print(s2)       # Welcome to Python
print(s3)       # Multi-line output

Python treats everything between the quotes as text data, regardless of its content.

String Immutability

Strings are immutable, meaning once a string is created, its content cannot be changed directly. Any modification creates a new string instead of altering the original one.

PYTHON 
s = "Python"
# s[0] = 'J'  # ‚ùå This will raise TypeError
#               'str' object does not support item assignment

s = "Jython"  # ‚úÖ This creates a new string object
print(s)      # Jython
üí° Key Concept

Strings are character sequences enclosed in quotes. They form the backbone of text processing, message display, and data formatting tasks in Python. Immutability ensures data safety and consistency.

Properties of Strings

Python strings come with several important properties that define their behavior:

Characters maintain fixed positions

Cannot be modified in place

üîÑ Iterable

1. Ordered (Indexing)

Each character has a fixed index position, starting from 0 for the first character. Indexing allows direct access to specific characters.

PYTHON 
s = "Interview"
print(s[0])     # I (first character)
print(s[4])     # r
print(s[-1])    # w (last character)
print(s[-2])    # e (second from last)

Indexes can be negative, where -1 represents the last character, -2 the second-to-last, and so on.

2. Immutable

Once created, a string cannot be modified in place. Any operation that seems to change a string actually creates a new string object.

PYTHON 
text = "Hello"
# Attempting to modify
# text[0] = "h"  # ‚ùå TypeError

# Creating a new string instead
new_text = "h" + text[1:]
print(new_text)     # hello
print(text)         # Hello (original unchanged)

3. Iterable

A string can be iterated character by character using loops.

PYTHON 
for ch in "Python":
    print(ch, end=" ")  # P y t h o n

# Using enumerate for index and character
for index, char in enumerate("AI"):
    print(f"Index {index}: {char}")

4. Supports Slicing

We can extract parts of a string using slicing syntax: s[start:end:step]

PYTHON 
s = "Interview"

print(s[2:6])       # terv
print(s[:5])        # Inter (from start to index 5)
print(s[5:])        # view (from index 5 to end)
print(s[::2])       # Itrir (every 2nd character)
print(s[::-1])      # weivretnI (reverse)
‚úÖ Summary

Strings are ordered, immutable, and iterable sequences. With slicing, we can extract and manipulate sections efficiently while maintaining data safety.

Basic Operations on Strings

Concatenation Repetition Membership

1. Concatenation (+)

Joining two or more strings together using the + operator.

PYTHON 
first = "Google"
second = " Cloud"
result = first + second
print(result)           # Google Cloud

# Multiple concatenations
full_name = "John" + " " + "Doe"
print(full_name)        # John Doe

# Using += operator
message = "Hello"
message += " World"
print(message)          # Hello World

2. Repetition (*)

Repeat a string multiple times using the * operator.

PYTHON 
print("AI " * 3)            # AI AI AI
print("=" * 20)             # ====================
print("-" * 10)             # ----------

# Creating patterns
print("*" * 5)              # *****
print(("*-" * 5))           # *-*-*-*-*-

3. Membership Testing (in / not in)

Check if a substring exists using in or not in.

PYTHON 
print("A" in "AI")              # True
print("Z" not in "AI")          # True

# Case-sensitive checking
print("python" in "Python")     # False (case matters!)
print("Python" in "Python")     # True

# Practical example
email = "user@example.com"
if "@" in email and "." in email:
    print("Valid email format")  # Valid email format

4. Length (len())

PYTHON 
text = "Python Programming"
print(len(text))            # 18

# Empty string
print(len(""))              # 0

# With spaces
print(len("  hello  "))     # 9 (spaces count!)
üí° Summary

Concatenation, repetition, and membership checks are fundamental string operations used in loops, data creation, and condition-based processing.

Search, Replace & Count Methods

Method Description Example
find() ‚òÖ Returns lowest index; -1 if not found "hello".find("l") ‚Üí 2
rfind() Returns highest index "hello".rfind("l") ‚Üí 3
index() Like find() but raises ValueError "hello".index("l") ‚Üí 2
rindex() Reverse of index() "hello".rindex("l") ‚Üí 3
count() ‚òÖ Counts substring occurrences "banana".count("a") ‚Üí 3
replace() ‚òÖ Replaces substring "data".replace("a", "o") ‚Üí 'doto'
startswith() ‚òÖ Checks prefix "machine".startswith("mach") ‚Üí True
endswith() ‚òÖ Checks suffix "learning".endswith("ing") ‚Üí True

Detailed Examples

PYTHON 
# Finding substrings
text = "Python Programming"
print(text.find("Pro"))         # 7
print(text.find("Java"))        # -1 (not found)

# Counting occurrences
sentence = "google cloud platform"
print(sentence.count("o"))      # 3
print(sentence.count("cloud"))  # 1

# Replacing text
s = "data science data"
print(s.replace("data", "AI"))          # AI science AI
print(s.replace("data", "AI", 1))       # AI science data (1 replacement)

# Checking prefixes and suffixes
filename = "document.pdf"
if filename.endswith(".pdf"):
    print("PDF file detected")  # PDF file detected

url = "https://google.com"
if url.startswith("https://"):
    print("Secure connection")  # Secure connection
Interview Tip

The replace() and find() methods are among the most used in industry work for text analysis, pattern searches, and data cleanup pipelines. Remember that find() returns -1 when not found, while index() raises a ValueError.

Splitting, Joining & Line Handling

Method Description Example
split() ‚òÖ Splits into list "a,b,c".split(",") ‚Üí ['a','b','c']
rsplit() Splits from right "a b c".rsplit(" ", 1) ‚Üí ['a b','c']

Comprehensive Examples

PYTHON

# Splitting by lines
multiline = "hello\nworld\npython"
lines = multiline.splitlines()
print(lines)                    # ['hello', 'world', 'python']

# Joining elements
words = ["AI", "Machine", "Learning"]
print(" ".join(words))          # AI Machine Learning
print("-".join(words))          # AI-Machine-Learning
print("".join(words))           # AIMachineLearning

Practical Application: CSV Processing

PYTHON 
# Processing CSV-like data
data = "John,25,Engineer\nAlice,30,Designer\nBob,28,Developer"

# Split into lines
records = data.splitlines()

# Process each record

    fields = record.split(",")
    name, age, role = fields
    print(f"{name} is a {age}-year-old {role}")

# Output:
# John is a 25-year-old Engineer
# Alice is a 30-year-old Designer
# Bob is a 28-year-old Developer

String Formatting Styles

% Formatting .format() f-Strings Template

A) Old % Formatting (Legacy)

Simple but outdated style used in legacy Python code.

PYTHON 
name, age = "Alice", 25
print("My name is %s and I am %d years old." % (name, age))
# Output: My name is Alice and I am 25 years old.

# Format specifiers
print("Pi = %.2f" % 3.14159)    # Pi = 3.14
print("Hex: %x" % 255)          # Hex: ff

B) str.format() Method

More powerful and readable than % formatting:

PYTHON 
# Positional arguments
print("My name is {} and I am {}".format("Bob", 30))
# Output: My name is Bob and I am 30

# Named arguments
print("My name is {n} and I am {a}".format(n="Bob", a=30))

# Index-based
print("{0} {1} {0}".format("Hello", "World"))
# Output: Hello World Hello

# Formatting numbers
print("Price: ${:.2f}".format(19.99))   # Price: $19.99

C) f-Strings (Modern & Recommended) ‚≠ê

Introduced in Python 3.6, f-strings are fast, readable, and allow inline calculations.

PYTHON 
name, age = "Charlie", 35
print(f"My name is {name} and I am {age}")
# Output: My name is Charlie and I am 35

# Inline expressions
print(f"Next year I'll be {age + 1}")
# Output: Next year I'll be 36

# Calling functions
print(f"Uppercase: {name.upper()}")
# Output: Uppercase: CHARLIE

# Mathematical operations
x, y = 10, 3
print(f"{x} + {y} = {x + y}")
# Output: 10 + 3 = 13

D) Template Strings (Safe for User Input)

Safe for user input, as they prevent code injection.

PYTHON 
from string import Template

temp = Template("Hello $name, balance: $$${amount}")
result = temp.substitute(name="Deepak", amount=5000)
print(result)
# Output: Hello Deepak, balance: $$$5000

E) Advanced f-String Formatting

PYTHON 
pi = 3.14159
print(f"Pi = {pi:.2f}")         # Pi = 3.14
print(f"Pi = {pi:.4f}")         # Pi = 3.1416

# Number formatting
print(f"Binary: {10:b}")        # Binary: 1010
print(f"Hex: {255:x}")          # Hex: ff
print(f"Percent: {0.75:.1%}")   # Percent: 75.0%

# Alignment
print(f"{'left':<10}|")         # left      |
print(f"{'center':^10}|")       #   center  |
print(f"{'right':>10}|")        #      right|
Interview Tip

f-strings are the most efficient way to format text dynamically in Python. They're faster than both % formatting and .format(), and they're the most readable. Always prefer f-strings in modern Python code (Python 3.6+).

4. Anagram Check

5. Remove Duplicates (Preserve Order)

PYTHON 
s = "programming"

seen = set()

for char in s:
    if char not in seen:
        result += char
        seen.add(char)

print(result)                   # progamin

6. Count Vowels and Consonants

PYTHON 
s = "Hello World"
vowels = "aeiouAEIOU"

print(f"Consonants: {consonant_count}") # Consonants: 7

7. Title Case Conversion (Manual)

PYTHON

8. String Compression

PYTHON 
s = "aaabbccccaa"
result = ""
count = 1

for i in range(1, len(s)):
    if s[i] == s[i-1]:
        count += 1
    else:
        result += s[i-1] + str(count)
        count = 1

result += s[-1] + str(count)
print(result)                   # a3b2c4a2
